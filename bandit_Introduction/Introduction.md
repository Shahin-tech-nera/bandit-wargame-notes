# Introduction

When I first started exploring the world of cybersecurity, one of the biggest challenges I faced was finding platforms where I could practically apply the theoretical knowledge I had learned. After some research, I discovered my first solution: the [OverTheWire Bandit Game](https://overthewire.org). I simply typed "bandit game" into Google, and the first result led me to the website – marking the beginning of my hands-on learning journey.  

Previously, I was using Windows 10. However, on October 15, 2025, I received a notification that support for Windows 10 on my laptop would end. Since Windows 11 would not be supported on my device, I decided to switch my operating system.  

To continue learning cybersecurity effectively, I chose **Ubuntu Linux**. I can honestly say that I do not regret this decision. Ubuntu has provided me with a comfortable practical environment and made my learning process much more efficient. From now on, all my game walkthroughs will be presented using Linux. Windows users can still follow along by connecting to a virtual Linux environment through WSL.
